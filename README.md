# Library Management System (CRUD Application)

## Description
This is a simple Library Management System built using HTML, CSS, and JavaScript.  
The application demonstrates CRUD operations (Create, Read, Update, Delete) on book records.

Data persistence is achieved using the browser's LocalStorage.

## Features
- Add new books
- View all books
- Update existing book details
- Delete books
- Search books by title or author

## Technologies Used
- HTML
- CSS
- JavaScript
- LocalStorage

## Note
This is a frontend-only application created for academic purposes.
